# Ejecutar en desarrollo

1. Clonar el repositorio

2. Instalar dependencias en el repositorio
```
npm install
```

3. Instalar @nestjs/cli de manera global
```
npm install -g @nestjs/cli
```

4. Levantar la base de datos
```
docker-compose up -d
```
